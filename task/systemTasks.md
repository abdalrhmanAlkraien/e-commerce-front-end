## Notes & Conventions

### Task Status Definitions

- **PENDING**: Task not started, waiting in queue
- **IN_PROGRESS**: Currently being worked on (one at a time)
- **COMPLETED**: Finished, tested, working correctly
- **BLOCKED**: Waiting on dependency or external factor
- **FAILED**: Completed but has bugs/issues, needs fixing

### Token Tracking

- **Input Tokens**: Tokens sent to Claude (context)
- **Output Tokens**: Tokens generated by Claude (response)
- **Total Tokens**: Input + Output
- **Cost**: (Input/1M × $3) + (Output/1M × $15)

### Fix Tracking

- **Fix Count**: Number of times task was fixed after completion
- **Fix Tokens**: Additional tokens spent fixing issues
- **Fix Cost**: Additional cost incurred by fixes

### Estimated vs Actual

- **Estimated**: Pre-calculated expectations
- **Actual**: Real values after completion
- **Variance**: Actual - Estimated (for learning)
